# weather
Projeto feito para etapa 5, desafio pratico front-end no Letras.com
Consiste em um web software que pega sua geolocalizaçao ou entao um local no qual queira consultar, e consulta o clima, tanto no momento quanto nos proximos 5 dias,
com opçoes de linguagem em pt_br, en ou sp, consumindo 2 apis, uma do google e outra do openWeather

  <h2 id="sobre">💻O projeto💻</h2>
  Projeto feito durante o processo seletivo do [Letras](https://letras.com)
  E uma aplicação que exibe os dados climáticos da região ou das cidades escolhidas, com funçoes de:
  geolocalizaçao, autocomplete, previsao para os proximos 5 dias, medidas em °C ou °F e 3 opçoes de idiomas. 
  Feito com React, typescript, JS e HTML.

<h2 id="aplicacao">⚙️Rodando a aplicação⚙️</h2>

Requisitos:  [Git](https://git-scm.com/),  [Node.js](https://nodejs.org/en/),  [Yarn](https://yarnpkg.com/). Além disto é bom ter um editor para trabalhar com o código como  [VSCode](https://code.visualstudio.com/).

```
# Clone este repositório
$ git clone https://github.com/wesleywisch/Clima.git

# No terminal, acesse a pasta do projeto
$ cd weather

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start

# O app estará disponível no seu browser pelo endereço - http://localhost:3000
```

Obs: precisa pegar a sua chave api do [openweather](https://openweathermap.org/api) e a do [GoogleCloud](https://cloud.google.com/apis)
